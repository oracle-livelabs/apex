# Import the Social Media Application

## Introduction

In this lab, you import the Social Media application into your APEX workspace. You use this imported app to integrate with the OCI Vision service.

Estimated Time: 15 minutes

### Objectives

In this lab, you learn how to:

- Import the Social Media application into your APEX workspace.
- Add columns to a table using Object Browser to store the AI response.

## Task 1: Import the Social Media Application

1. Download the [Social Media application](files/APEX_Social_Media_HOL_FINAL.sql).

2. In the left navigation bar, hover over the **App Builder** icon.

    Click **Import**.

    ![Click Import in App Builder](images/app-import.png " ")

3. The Import dialog is displayed. Perform the following steps:

    - Drag and drop the file, or navigate to the file.

    - Click **Next**.

    ![Upload file and click Next](images/import-sql-file.png " ")

4. Accept the defaults.

    Click **Import Application**.

    ![Click Import Application](images/import-loaded.png " ")

5. Click **Install Supporting Objects**.

    ![Click Install Supporting Objects](images/install-app2.png " ")

6. Verify that the application's supporting objects were installed successfully.

    ![Supporting Objects Installed](images/supporting-objects-installed.png " ")

## Task 2: Add Columns to the Table Using Object Browser

To store the output generated by the OCI Vision service, add two columns to the **SM\_POSTS** table.

To add two new columns to the **SM\_POSTS** table:

1. In the left navigation bar, hover over the **SQL Workshop** icon.

    Click **Object Browser**.

    ![Navigate to SQL Workshop and select Object Browser](images/sql-work2.png " ")

2. In the object tree, expand Tables and select **SM\_POSTS** table.

    On the columns tab, click **Add Column**.

    ![Select SM_POSTS and click Add Column](images/add-column.png " ")

3. In the Add Column wizard, enter the following:
      - Column: **AI\_OUTPUT**
      - Data Type: **VARCHAR2**
      - Length: **1000**

      Click **Apply**.

    ![Enter AI_OUTPUT column details](images/ai-output-column.png " ")

4. Click **Add Column** and enter the following:

    - Column: **AI\_OUTPUT\_TD**
    - Data Type: **VARCHAR2**
    - Length: **4000**

    Click **Apply**.

    ![Enter AI_OUTPUT_TD column details](images/ai-output-td-column.png " ")

## Summary

You now know how to import an Oracle APEX application and add columns to a table.
